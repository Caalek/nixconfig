# NixOS Configuration

## Workflow: New Machine Setup

### 1. Boot ISO → Clone repo → Partition with Disko

```bash
# Boot the NixOS ISO, then:
nix-shell -p git
git clone <your-repo> /tmp/nixconfig

# Partition, format, mount (will prompt for LUKS passphrase)
sudo nix run github:nix-community/disko -- \
  --mode disko /tmp/nixconfig/disko-config.nix

# Generate hardware config
nixos-generate-config --root /mnt
cp /mnt/etc/nixos/hardware-configuration.nix /tmp/nixconfig/

# Install
nixos-install --flake /tmp/nixconfig#nixos
reboot
```

### 2. First boot → Symlink config → Generate age key

```bash
# Symlink this repo to /etc/nixos
git clone <your-repo> ~/nixconfig
sudo mv /etc/nixos /etc/nixos.bak
sudo ln -s ~/nixconfig /etc/nixos

# Generate a dedicated age key for agenix
sudo age-keygen -o /etc/age/key.txt
# ^ Save the public key output — you'll need it to encrypt secrets for this machine
```

### 3. On your main machine → Add this host's age key to secrets

```bash
# Add the new public key as a recipient to existing secrets
# (agenix re-encrypts for all recipients already in the file)
agenix -e secrets/borg-passphrase.age -i ~/.ssh/id_ed25519

# Commit the updated .age file
git add secrets/ && git commit -m "secrets: add new host"
```

### 4. On the new machine → Pull + Rebuild

```bash
git pull
sudo nixos-rebuild switch --flake /etc/nixos#nixos
```

Agenix decrypts secrets at build time using `/etc/age/key.txt`. No further setup needed.

## Daily Use

```bash
# Apply system + user changes
nixos-rebuild switch --flake /etc/nixos#nixos

# Apply only user config (home-manager, no sudo)
home-manager switch --flake /etc/nixos#vm
```

## Manage Secrets with Agenix

Encrypted secrets live in `secrets/`. They're decrypted at build time by agenix.

### Add a new secret

```bash
# Encrypt a value for the machine's age public key
echo -n "my-secret-value" | age -r age1abc... --armor > secrets/my-secret.age
```

In `configuration.nix`:
```nix
age.secrets.my-secret = {
  file = ./secrets/my-secret.age;
  owner = "root";
  group = "root";
};
# Access via ${config.age.secrets.my-secret.path}
```

## Structure

```
nixconfig/
├── flake.nix                    # Inputs: nixpkgs, home-manager, llm-agents, agenix
├── configuration.nix            # System config (boot, services, users, secrets)
├── hardware-configuration.nix   # Auto-generated by nixos-generate-config
├── disko-config.nix             # Disk layout for install
├── home.nix                     # Home-manager (user packages, dotfiles, dconf, gtk)
├── secrets/
│   └── borg-passphrase.age      # Encrypted borg backup passphrase
└── README.md
```
